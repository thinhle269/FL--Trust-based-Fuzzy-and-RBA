# RBA Fuzzy + Federated Learning (ANN-BN, FedAvg)

## Quick start
```bash
python -m venv .venv && source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt

# 1) Build fuzzy trust + silver labels (lấy file excel co label)
python build_labels.py --xlsx data/rba_test.xlsx --out data/rba_fuzzy_labels.csv

# 2) Run experiments (centralized LR + FL ANN-BN + ablations + figures)
python run_experiments.py --csv data/rba_fuzzy_labels.csv --out outputs
